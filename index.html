Guía para el tutor

El Triple Method combina memoria visual, movimiento, pausas estructuradas y práctica digital para reducir frustración y construir confianza.

Descargar grid 12×12 Ver metodología completa en GitHub

Principios base

01Regla 20/15

Sesiones de máximo 20 minutos, seguidas por 15 minutos de pausa física.

02Manos ocupadas

Un fidget en la mano no dominante ayuda a canalizar energía durante la escritura.

03Confianza progresiva

Avanza por dificultad lógica: 1, 2, 5, 10; luego 3, 4, 6, 11; finalmente 7, 8, 9, 12.

Secuencia Triple Method

AGrid

Completar el grid imprimible mientras se dice cada operación en voz alta.

BLanzamiento

Responder de memoria mientras se lanza y recibe un objeto suave.

CMatPro App

Practicar las tablas del día con distractores inteligentes y revisión de fallos.

Cómo se juega

1Practica las tablas

El tutor o jugador elige tablas, dificultad, número de preguntas y tiempo. Si el temporizador está en 0, la ronda no tiene límite de tiempo.

2Maxitablas Pro

Ronda completa de 144 preguntas con todas las multiplicaciones desde 1×1 hasta 12×12.

3Revisión de fallos

Al terminar, revisa solo errores y omitidas para enfocar la siguiente sesión.

Acerca de este proyecto

MatPro nació para ayudar a aprender con confianza.

MatPro es un sistema de aprendizaje humano, ADHD-friendly, creado para ayudar a niños a memorizar las tablas de multiplicar con estructura, movimiento y práctica digital.

El proyecto nació del deseo de un padre de ayudar a su hijo Momo a superar la frustración con matemáticas, combinando apoyo familiar, principios de terapia ocupacional y desarrollo iterativo.

MatPro es la base previa a DivisionPro, donde esas habilidades se aplican a la división larga.

Developed with ❤️ for my son and students everywhere.
